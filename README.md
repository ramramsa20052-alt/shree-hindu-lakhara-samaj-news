# Shree Hindu Lakhara Samaj Digital News Website

A modern, professional, and dynamic digital news portal for the Shree Hindu Lakhara Samaj.

## Features
- Dynamic news sections (India, Rajasthan, Religion, Tech, etc.)
- Event and Notice management
- Society and Member directories
- Gallery and Media management
- Admin Dashboard for content management

## Technologies
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- MDB5 / Bootstrap 5
- Font Awesome 6

## Project Structure
- `index.html`: Main entry point
- `assets/`: CSS and JS files
- `pages/`: Individual news and category pages
- `pages/admin/`: Backend management interface
